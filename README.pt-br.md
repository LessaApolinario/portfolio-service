# Portfolio Service

![Architecture](https://img.shields.io/badge/Architecture-Microservice-blue)
![Status](https://img.shields.io/badge/Status-In%20Development-green)
![License](https://img.shields.io/badge/License-MIT-lightgrey)

Este projeto é um **microserviço de domínio** responsável por gerenciar dados relacionados a portfólios.

Ele contém a lógica de negócio principal para manipulação de portfólios de usuários, ativos e operações relacionadas, sendo projetado para ser **independente do frontend**.

Este serviço é consumido pelas camadas de Backend for Frontends (BFF) dentro da arquitetura.
